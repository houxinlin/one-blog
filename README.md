## 项目部署（Web）

1. 下载
```
git clone https://github.com/houxinlin/OneBlog.git
npm install
```

2. 修改项目配置

替换成自己服务器地址
```
.env.production
```

3. 运行
```
npm run serve
```

4. 打包
```
npm run build
```
